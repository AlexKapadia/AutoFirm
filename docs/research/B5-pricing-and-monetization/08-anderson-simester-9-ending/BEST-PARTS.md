# BEST PARTS — Charm / 9-Ending Pricing

## ADOPT
- **A behavioral price-presentation layer as a SEPARATE, optional, evidence-gated step.** Build
  implication: after the price-level engine outputs a value/elasticity-optimal price, an optional
  `presentation` module may apply charm-ending / framing rules -- but only in contexts where the
  evidence supports it (B2C retail / e-commerce / DTC in the B12 panel), and never for
  enterprise/B2B negotiated pricing. The optimal *level* is set by EVC + elasticity FIRST;
  presentation tweaks are a thin, reversible layer on top, not the pricing logic.
- **Charm endings as an A/B-testable hypothesis, not a hardcoded rule.** Because the effect is
  context-dependent (PLOS ONE null), AutoFirm treats 9-endings as a hypothesis to validate per
  client via experiment (ties to evidence/measurement), not a universal default. Build implication:
  presentation rules carry a confidence/`requires_validation` flag.
- **Reference-price / framing awareness in value communication.** The "image effect" and left-digit
  findings inform how prices and discounts are *presented* in the value-communication artifact
  (anchor to a reference price; frame as a gain) -- ties to prospect theory (source 09).

## REJECT / REJECT-IN-CONTEXT
- **Reject charm pricing for B2B/enterprise/value-based negotiated deals.** It signals "consumer
  bargain," undermining a premium value-based position -- contradicts EVC-anchored pricing (source
  02). Apply only to mass-market B2C.
- **Reject treating the 9-ending effect as universal or large.** The PLOS ONE null (2022) shows it
  is weak/absent in some (notably online) settings; never rely on it as a primary revenue lever.
- **Reject deceptive framing** (fake reference prices, "was/now" without basis) -> B10 legal gate;
  many jurisdictions regulate reference-price advertising. Fail-closed.

## Build implication (concrete)
Adds an **optional, industry-gated, validation-flagged price-presentation layer** to L2.B5, strictly
downstream of the value/elasticity price-level engine. Demonstrates AutoFirm's separation of
*correct pricing* (deterministic core) from *presentation psychology* (optional, evidence-gated, A/B
tested) -- and its honesty about a contested effect (carries the contradicting PLOS ONE evidence).
