## Key claims (faithful)
1. **Answer-first (top-down):** start with the **single governing conclusion/recommendation** (the "governing thought"), then support it with grouped key arguments, then back each argument with data. "Never bury the lead." Inverts the usual build-up-to-conclusion order.
2. **Pyramid structure:** Main idea -> key supporting arguments (the logical pillars, grouped by theme) -> data/evidence under each. Each level summarizes the level below.
3. **MECE** (Mutually Exclusive, Collectively Exhaustive) grouping: the supporting arguments under any point must not overlap and must together cover the whole.
4. **Action titles / message titles:** each slide's title states the slide's **specific takeaway message** (a full sentence asserting the point), not a topic label ("Revenue grew 23% on enterprise expansion", not "Revenue"). Reading only the titles top-to-bottom should reconstruct the storyline.
5. Rationale: executives want the destination first then the supporting logic; the structure serves comprehension under time pressure.
