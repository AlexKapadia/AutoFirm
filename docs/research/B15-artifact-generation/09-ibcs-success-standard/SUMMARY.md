## GRADE tier
**High (primary open standard).** The standard of record for IBCS; governed, openly published. Practitioner standard (not an empirical study), so efficacy is by adoption/expert consensus, corroborated by Tufte (source 07) on the underlying perceptual principles.

## Key claims (faithful)
1. **SUCCESS rules** — business communication meets IBCS if it follows the seven-rule SUCCESS formula:
   - **S — SAY:** convey a (single, clear) message.
   - **U — UNIFY:** apply **semantic notation** (consistent visual vocabulary — same meaning always shown the same way).
   - **C — CONDENSE:** increase information density.
   - **C — CHECK:** ensure visual integrity (no misleading scales/truncated axes).
   - **E — EXPRESS:** choose proper visualization.
   - **S — SIMPLIFY:** avoid clutter (aligns with Tufte's chartjunk removal).
   - **S — STRUCTURE:** organize content (MECE-style; aligns with Minto).
2. **Three pillars:** **Conceptual** (organize content to get the message across), **Perceptual** (which visualizations to use), **Semantic** (the IBCS notation — standardized terminology, descriptions, dimensions, analyses, indicators).
3. **Semantic notation (Hichert):** a consistent visual language — e.g. fixed conventions for actuals vs plan vs forecast vs previous-year, for variances (absolute/relative), and for scaling — so any reader decodes a chart the same way across all reports.
4. Governance: Creative Commons; IBCS Association maintains and certifies.

## Reproducibility note
