## Key claims (faithful)
1. **Message drives the chart, not the data.** "Choosing the right chart form starts with a clear message." Articulate the single point first; the chart reflects *that point*, not the raw measures.
2. **Five comparison types -> five chart families.** Zelazny maps **five data-relationship/comparison types** to chart forms:
   - **Component** (share of a whole, "percentage of") -> **pie chart**.
   - **Item** (ranking/comparison among items) -> **bar chart**.
   - **Time series** (change over time, "rose/fell/grew") -> **column chart** (or line for many points).
   - **Frequency distribution** (how many items fall in ranges) -> **histogram** / column.
   - **Correlation** (relationship between two variables, "relates to / varies with") -> **scatter/dot** (or paired bar).
3. **Keyword-to-chart heuristic:** the message's wording signals the type — "rising/grew" -> line/column; "largest percentage/share" -> pie; "ranks/compares" -> bar; "relates to/correlates" -> scatter.
4. Principles: simplicity, clarity, relevance; one message per chart.
