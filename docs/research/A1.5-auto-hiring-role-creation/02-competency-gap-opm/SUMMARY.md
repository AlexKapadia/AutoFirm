## What it is
OPM's standing methodology for **competency gap analysis**: the disciplined practice US federal
agencies use to identify the difference between the competencies a workforce *needs* (target
proficiency for an occupation/mission) and the competencies it *currently has* (assessed
proficiency), focused on **mission-critical occupations (MCOs)**.

## Key claims (with locators)
- **Gap = target proficiency - current proficiency**, per competency, per occupation. A gap
  analysis compares **required competencies** for a role/occupation against the **current
  competency proficiency** of incumbents to surface shortfalls and prioritise interventions.
- **Mission-critical occupations** are identified first, so gap analysis is targeted at the roles
  most consequential to the organisation's mission - not run uniformly across all roles.
- **Closed loop:** identify required competencies -> assess current proficiency -> compute the
  gap -> design interventions (hire, train, reassign) -> re-assess. This mirrors source 01's
  milestone 5 (bridge solutions) and the iterate-to-perfection loop.
