# Analysis-only dependencies for the output_review evidence showcase.
#
# CLAUDE.md §3.10 dependency boundary (BINDING): these plotting / stats / diagram
# libraries are used ONLY by the scripts under evidence/output_review/ to render the
# graphs and flow diagrams. They are NEVER imported by any runtime module under
# src/autofirm/ — the import-linter contract "Runtime code must not import
# analysis-only libraries" (2 kept / 0 broken) fences them out of the runtime
# closure. Install them into a venv ONLY to regenerate the showcase:
#
#     pip install -r evidence/output_review/requirements-analysis.txt
#
# The harness itself (golden_set/) uses ONLY the standard library + the runtime
# autofirm package, so the measured numbers can be reproduced without any of these.
#
# (These mirror the repo-level `analysis` optional-dependency extra in pyproject;
# this standalone file keeps the output_review showcase self-contained and separable
# from the runtime package, and never touches the shared runtime dependency block.)
